## 🔍 Creative Video Search Filters

### Time & Duration Filters

| Filter | Description |
|--------|-------------|
| **Quick Watch** | Videos under 5 minutes |
| **Coffee Break** | 5-15 minutes (perfect for a break) |
| **Deep Dive** | 30+ minutes for in-depth learning |
| **Binge-Worthy** | Series/playlists with 10+ episodes |
| **Recently Added** | Indexed in the last 7 days |
| **Throwback** | Videos published 5+ years ago |
| **Fresh Content** | Published in the last 30 days |

### Engagement & Quality Filters

| Filter | Description |
|--------|-------------|
| **Highly Rated** | Like ratio > 95% |
| **Hidden Gems** | High quality score but low view count |
| **Viral Hits** | 1M+ views |
| **Discussion Starters** | High comment-to-view ratio |
| **Controversial** | High engagement but mixed sentiment |

| Filter | Description |

| **4K Available** | High-resolution content |
